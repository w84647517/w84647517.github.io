# 基本类型

TIP

类型需要小写

类型 举例 描述
number 1,-33,2.55 任意的数字
string 'hi',"hello",hello 任意字符串
boolean true,false 布尔值true或false
字面量 其本身 限制变量的值就是该字面量的值
any * 任意类型
unknown * 类型安全的any
void 空值（undefined） 没有值或者undefined
never 没有值 不能为任何值
object {name:"wsen",age:25,sex:'男'} 任意的js对象
array [1,2,3,4] 任意的js数组
tuple [4,2] 元素，TS新增类型，固定长度数组
enum enum{a,b} 枚举，TS中新增类型
类型注意事项

能用unknown不用any

any可以赋值给任何的变量，unknown不可以直接赋值给别的类型变量

let s:string;
let a:any;
let u:unknown;
u='hello'
a=10;
s=a  //使用any不会报错 
s=u //使用unknown会报错 
if(typeof u ==='string'){
  s=u //判断完类型再赋值不会报错
}
//类型断言
s = u as string | <string>u
